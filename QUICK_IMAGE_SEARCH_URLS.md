## Direct Download Instructions

### Using Browser (Recommended)

1. **Go to Unsplash.com**
2. **Search using the links above**
3. **Click on an image you like**
4. **Click the download button**
   - Choose "Download" for standard size
   - Choose "Download free" for free version
5. **Save to your images folder**

### Using Image Downloader Extension

1. **Install "Image Downloader" extension for Chrome**
2. **Navigate to Unsplash search page**
3. **Click the extension icon**
4. **Select images to download**
5. **Choose download location**

### Batch Download Script (Bash/Mac/Linux)

```bash
# Create directory structure
mkdir -p public/images/tours/{golden-ring,chimgan,bukhara-families,desert,tashkent-modern,culinary-craft,complete-silk-road,khiva-fortresses}
mkdir -p public/images/cities/{tashkent,samarkand,bukhara,khiva,fergana,namangan,andijan,nukus,termez,gulistan,jizzakh,kokand,navoi}

# Note: You'll still need to manually download and place images
```

### Windows PowerShell

```powershell
# Create directory structure
New-Item -ItemType Directory -Force -Path "public\images\tours\golden-ring"
New-Item -ItemType Directory -Force -Path "public\images\tours\chimgan"
```

## Image Naming Convention

**Format**: `{city-or-tour}-{specific-location-or-feature}-{descriptor}.{ext}`

**Examples**:
- `samarkand-registan-square-sunset.webp`
- `bukhara-kalyan-minaret-daylight.jpg`
- `khiva-ichan-kala-walls-sunrise.webp`
- `tashkent-metro-station-interior.jpg`
- `uzbekistan-plov-cooking-class.webp`

## Alt Text Template

**Format**: "{Location} - {Description} - {Context}"

**Examples**:
- "Registan Square at sunset with three illuminated madrasahs in Samarkand, Uzbekistan"
- "Tourists hiking in Chimgan Mountains near Tashkent, showing snow-capped peak"
- "Kalyan Minaret standing tall in Bukhara's historic city center, Uzbekistan"
- "Traditional Uzbek plov cooking class in Samarkand with instructor and ingredients"
- "Ichan-Kala walled city in Khiva at golden hour, UNESCO World Heritage site"
