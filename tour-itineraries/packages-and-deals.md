1. **Choose your package** (A-E or custom)

   - Which tours you want

   - Any special requests

3. **We create custom proposal** (2-3 days)
   - Check availability for all components

7. **Receive complete documentation** (2 weeks before departure):
   - Combined itinerary
   - All confirmations
   - Pre-tour preparation guide specific to your package
   - Extensions documentation

## FREQUENTLY ASKED QUESTIONS ABOUT PACKAGES

**Q: Can I mix and match any tours?**
A: Almost! Most combinations work. We'll check logistics and advise if any don't make sense.

**Q: How much rest time between tours?**
A: We recommend minimum 1 full rest day. For physically demanding tours (Archaeology + Adventure), suggest 2-3 days rest between.

**Q: Can I stay in same hotel room between tours?**
A: Usually yes, especially in Tashkent, Samarkand. We'll confirm based on availability.

**Q: What if I want to add a third tour?**
A: Absolutely! We'll create custom 30+ day package. Discounts increase with length.

**Q: Do I get the same guide for both tours?**
A: Depends on tours. For Archaeology + Photography = NO (different specialist guides). For Culinary + Textile = YES, possible (same region guide). We'll advise.

**Q: Are flights included between tour segments?**
A: Package includes all transport between segments. If flights needed = included. If train/private vehicle = included.

**Q: Can I book extension later (after arriving)?**
A: Depends on availability, but usually YES (extension operations are simpler to arrange last-minute than main tours).

**Q: What if I finish first tour and realize I want to skip second?**
A: Unfortunately no refund (operational costs already committed). But you can:
- Transfer second tour to another person
- Possibly substitute different tour (if availability)
- Or just enjoy the experience you signed up for!

**Q: Can I book private departure for package?**
A: YES! All tours can be private. Package discounts still apply.
